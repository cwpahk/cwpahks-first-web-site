HTML

Hyper Text Markup Language ( English : Hyper Text Markup Language, HTML ) is the dominant markup language developed for displaying web pages . In addition, HTML provides a way to create structured documents with links, citations, and other items, as well as representing structural semantics for body text, such as headings, paragraphs, and lists .

it is written in the form of HTML elements with "tags" enclosed in angle brackets . HTML can contain or invoke scripts , such as JavaScript , which affect the behavior of HTML processing devices, such as web browsers , and CSS , which define the appearance and layout of body and other items . The W3C , co-leader of the HTML and CSS standards , recommends the use of CSS for clear and expressive markup.
